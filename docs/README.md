# SS Tickets - Sistema Piloto de Chamados Internos

**Empresa:** Smaniotto Solutions  
**Descrição:** Sistema de gerenciamento de tickets para suporte interno de empresas parceiras.

## Requisitos do MVP
- **Autenticação:** Login via e-mail e senha validados no PostgreSQL.
- **Perfis de Acesso:** Solicitante, Analista e Administrador.
- **Atributos do Ticket:** Título, Descrição, Prioridade

## Fluxo de Trabalho (Workflow)
1. Aguardando Atendimento
2. Em Atendimento
3. Aguardando Cliente
4. Respondido
5. Tratativa Interna
6. Resolvido
